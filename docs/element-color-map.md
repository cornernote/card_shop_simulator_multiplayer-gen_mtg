# Element To MTG Color Map

The generator maps game elements to MTG color identity as a scoring preference, not a hard rule. A card can still land outside the preferred colors when it wins on rarity, flavor, event fit, legendary/iconic value, or an explicit override.

| Game element | Preferred MTG colors |
| --- | --- |
| Fire | Red |
| Water | Blue |
| Grass | Green |
| Electric | Red or Blue |
| Insect | Green or Black |
| Rock | Red or Green |
| Earth | Green or Red |
| Animal | Green or White |
| Steel | Colorless preferred |
| Dragon | Red, Green, or Black |
| Psychic | Blue or White |
| Mystic | Black or Blue |
| Ice | Blue or White |

## Notes

- Explicit overrides can ignore this map.
- Artifact exceptions like `Black Lotus` and `Sol Ring` are allowed for iconic Old School spell-flavored slots.
- Colorless MTG creatures/artifacts are favored for Steel when the flavor also fits.
